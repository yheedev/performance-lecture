### 실행

```
// 다운로드
git clone https://github.com/yheedev/performance-lecture.git

cd performance-lecture
code .

// 의존성 설치
npm i

// 개발 서버 가동
npm run start

// json 서버 가동. npm run start 실행한 콘솔과 다른 콘솔에서 실행한 뒤에 종료하지 않을 것.
npm run server

// build & serve
npm run serve
```
